System Interactions

🧠 Memory → Learning

Episodic memories become training data. Each experience is extracted, generalized, and stored as procedural knowledge for future use.

⚡ State → Autonomy

Cognitive states modulate autonomy level. High curiosity triggers exploration mode; high frustration triggers caution mode.

📚 Learning → Memory

New skills and strategies are stored in procedural memory. Lessons learned from failures become part of semantic knowledge.
