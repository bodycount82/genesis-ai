Autonomy Mode

Give Genesis a goal and step away. It plans, executes tasks, browses the web, manages files, and works toward objectives on its own — using its full set of tools and memory.

The Loop

How autonomous work flows

Genesis doesn't follow a rigid script. It enters a continuous cycle of perception, planning, execution, and adaptation — repeating until the goal is achieved.

1

Analyze Goal

Breaks down the objective into manageable sub-tasks

2

Plan Actions

Creates an ordered sequence of steps to complete the goal

3

Execute

Performs each action using available tools and memory

4

Track Progress

Monitors what's done and what remains

5

Adapt

Adjusts the plan if obstacles are encountered

📖 Example: What happens when you give Genesis a goal

"Research the top 10 AI agent platforms and compare their pricing, features, and target audiences."

Step 1 Analysis: Genesis breaks this into: search for platforms, collect details for each, compare pricing, analyze features, identify target audiences.
Step 2 Planning: Creates an ordered plan — search web for platforms → visit top results → extract data → build comparison table → write summary.
Step 3 Execution: Searches multiple sources, fetches pages, extracts pricing tables, reads documentation, and compiles findings.
Step 4 Adaptation: Notices some sites require login — pivots to alternative sources. Finds incomplete data for one platform — marks it as "limited info available."
Step 5 Deliverable: Creates a structured comparison document with pricing tiers, feature matrices, and audience analysis. Summarizes key findings and recommendations.
Control

You're always in control

Autonomy doesn't mean uncontrolled. Genesis operates within boundaries you define, and you can monitor or intervene at any time.

Time limits: You set how long Genesis works autonomously before reporting back. It won't run indefinitely — it stops when the goal is complete or the time limit is reached.

Interrupt at any time: Send a message, switch modes, or close the application. Genesis stops immediately and reports what it was doing.

Visible progress: Watch Genesis work in real time. See each task as it's completed, review files as they're created, and read progress notes as they're written.

When to use Autonomy mode

  • Research tasks: "Find me information about X" — Genesis searches, reads, and synthesizes while you do something else.
  • Multi-step projects: "Build a competitor analysis" — Genesis creates the project, researches each competitor, and delivers a report.
  • Repetitive work: "Organize these files" or "Send follow-up emails to these contacts" — Genesis handles the grunt work.
  • Exploration: "Look into this topic and tell me what's interesting" — Genesis investigates broadly and shares discoveries.
