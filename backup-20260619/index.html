Capabilities

More than a chatbot

Genesis combines persistent memory, autonomous operation, and real-world tool use into a single application that runs locally on your computer.

⚙

Autonomous Operation

Genesis doesn't just wait for instructions. In Autonomy mode, it sets its own goals, pursues projects, and makes progress while you're away — then reports back what it accomplished.
💪

Biological Memory

Five interconnected memory tiers — episodic, semantic, procedural, working, and sensory — with real forgetting, reinforcement, and consolidation. Memories fade if unused; important ones persist.
💻

Desktop Control

Genesis can browse the web, control your mouse and keyboard, read files and PDFs, manage spreadsheets, send emails, and operate applications on your computer.
🎨

Simulated Moods

Internal state values for energy, curiosity, frustration, and satisfaction influence how Genesis approaches problems — not as rigid rules, but as advisory signals that shift over time.
📚

Project Management

Built-in project board with backlog, todo, in-progress, in-review, and completed columns. Genesis creates tasks, tracks progress, and maintains living documentation for every project.
🔒

Local & Private

Runs entirely on your machine. Connects to local LLMs via Ollama or llama.cpp, or API providers of your choice. Your data stays on your computer.
Three Modes

Built for different ways of working

Switch between direct conversation, hands-off autonomy, or focused project work — depending on what you need right now.

💬

Chat

Direct conversation. Ask questions, get answers, work together in real time. The most familiar way to interact with AI — but with the depth of a system that remembers everything.
⚪

Autonomy

Give Genesis a goal and step away. It plans, executes tasks, browses the web, manages files, and works toward objectives on its own — using its full set of tools and memory.
📈

Work

A structured mode for project-based work. Genesis breaks down complex tasks, creates a project board with milestones, and works through them systematically.
Memory Architecture

A memory system that works like a mind

Five tiers of memory, each serving a different purpose. Memories are encoded, strengthened by use, and allowed to fade when unused — just like human memory.

Episodic

Events and experiences from your interactions

Semantic

Facts and knowledge about the world

Procedural

Strategies, skills, and approaches that work

Working

Active context for current tasks

Sensory

Visual and perceptual data from the environment

Getting Started

How it works

Get up and running in minutes — no technical expertise required.

1

Download the installer

Get the latest version — a single executable file that sets everything up automatically.

2

Choose your backend

Select from local models via Ollama or llama.cpp, or use cloud APIs. The installer handles all dependencies.

3

Start chatting

Genesis launches into Chat mode — talk to it, give it tasks, or switch to Autonomy and let it work on its own.

4

Watch it grow

The more you use Genesis, the better it gets. Its memory strengthens with use, and it learns your preferences.
