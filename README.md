# Genesis Website — File Map

Static website for **Genesis** — an autonomous AI companion. Built as pure HTML/CSS/JS, deployed on GitHub Pages.

## Structure

```
web page for publishing/
├── index.html              # Landing page (hero, features, modes, memory, specs, download)
├── README.md               # This file
├── backup-20260619/        # Historical backup of 9 HTML files from June 19
│   └── (9 original HTML files)
└── ~100 standalone .html pages
```

Every page is self-contained: HTML + inline CSS + inline JS in a single file. No external dependencies, no build step, no framework.

## Technical Notes

- **Format**: Pure HTML5 with inline CSS and JavaScript — no external libraries, no build tools
- **Theme**: Dark mode (`#0a0a0f` background) with purple accent (`#7b6ff0`)
- **Responsive**: Mobile-friendly with media queries at 768px breakpoint
- **Animations**: CSS transitions + IntersectionObserver for scroll fade-ins
- **Navigation**: Fixed top nav with dropdown menus, consistent across all pages
- **Footer**: Standard site footer with links and copyright on every page
- **No server required**: Can be opened directly in a browser (`file://`) or served from any static host
