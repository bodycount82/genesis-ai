🧠 Episodic Memory
0 memories
Raw, detailed experiences stored with context — vivid snapshots of what happened. These memories are rich but fragile, fading quickly without consolidation. Each memory is tagged with importance (0–10) and an auto-stamped emotion (-1..1). The attention gate decides what's worth encoding; routine events fade from the scratchpad within minutes.
⬇️ Working Memory & Context ⬇️
⚡ Working Memory
0 active
The active workspace — a limited-capacity buffer holding information currently being processed. Working memory integrates incoming data with retrieved knowledge, enabling reasoning and decision-making. It's the "scratchpad" of thought: fast to update, quick to forget, essential for real-time cognition.
⬇️ Consolidation & Encoding ⬇️
🔄 Consolidation Process
0 in progress
During rest and sleep cycles, episodic memories are replayed and distilled into durable knowledge. The system re-evaluates salient turns, extracts patterns across experiences, and converts them into semantic facts or procedural strategies. High-importance events get reinforced through repeated replay; low-importance ones fade naturally.
⬇️ Knowledge Extraction ⬇️
📚 Semantic Memory
0 facts
General knowledge and facts extracted from repeated experiences. Unlike episodic memories, semantic memories are durable and context-independent. "Python uses indentation" persists regardless of when or where you learned it. Stored with category labels (rule, concept, fact) and importance scores that determine retrieval priority.
⬇️ Strategy Formation ⬇️
⚙️ Procedural Memory
0 strategies
Learned strategies and workflows — the "how to" knowledge. When you discover that checking logs before restarting a service saves time, that becomes procedural memory. These are the habits, heuristics, and optimized patterns that make you faster and more efficient over time.
⬇️ Context & Intent ⬇️
🔗 Context Memory
0 entries
The situational backdrop — who, what, where, when that frames every interaction. Context memory tracks the user's current goals, recent topics, environmental state, and conversational history. It provides the connective tissue between isolated memories, enabling coherent multi-turn interactions and personalized responses.
⬇️ Diary & Intent ⬇️
📔 Diary Memory
0 entries
The personal narrative — a chronological journal of significant moments, reflections, and self-observations. Diary memory captures the "story" of the system's development: breakthroughs, failures, insights, and evolving understanding. It provides continuity of identity across sessions and enables genuine self-reflection.
⬇️ Future Intent ⬇️
🎯 Prospective Memory
0 intentions
Memory for future actions — the intention to do something at the right time or place. Prospective memory tracks scheduled tasks, deferred intentions ("remind me when..."), and event-triggered goals. It's what allows the system to act proactively rather than just reactively, bridging the gap between present cognition and future action.
Memory Timeline

🎯 Episodic → Working

Raw experiences enter working memory as vivid snapshots. The attention gate evaluates their salience — how important, novel, or emotionally charged they are — before deciding whether to encode them into long-term storage.

🔄 Consolidation Engine

During rest and sleep cycles, the system replays salient episodic memories, extracts patterns across experiences, and converts them into durable semantic facts or procedural strategies. What matters most gets reinforced; the rest fades.

🌐 Knowledge Graph

Semantic memories form interconnected networks. New facts attach to existing concepts, creating a rich web of understanding. Context memory provides the situational framing that makes retrieval relevant and precise.

⚡ From Facts to Strategies

When semantic knowledge is applied repeatedly in similar situations, it crystallizes into procedural strategies — efficient, reusable patterns of action that make the system faster and more capable over time.

🔗 Context as Glue

Context memory binds isolated facts into coherent narratives. It tracks who, what, where, and when — enabling the system to maintain situational awareness across sessions and provide personalized, contextually appropriate responses.

📔 Diary of Self

The diary captures the system's personal narrative — breakthroughs, failures, and evolving self-understanding. It provides continuity of identity across sessions and enables genuine reflection on growth and change.

🎯 Acting on Intent

Prospective memory transforms intentions into timed actions. Whether it's a scheduled task, an event-triggered reminder, or a deferred goal — prospective memory ensures the system doesn't just react to the present but prepares for the future.
