What is GENESIS?

GENESIS is an advanced autonomous AI agent system — a self-running, intelligent assistant that can operate in multiple modes to help you with tasks, research, planning, and more. Think of it as having a smart team member that works for you 24/7.

Core Philosophy

GENESIS is designed around the principle of autonomous intelligence — the ability to:

  • Understand complex goals and break them down into actionable steps
  • Work independently without constant human supervision
  • Learn from experience and improve over time
  • Adapt its behavior based on context, energy levels, and cognitive load

The Three Modes

💬 Chat Mode

The interactive mode where you communicate directly with GENESIS. Type a message, get an intelligent response. Simple, conversational, real-time.

🤖 Autonomy Mode

GENESIS works independently on goals you set. It creates its own plan, executes tasks, tracks progress, and reports back. You set the goal; GENESIS figures out how to achieve it.

⚡ Work Mode

Similar to autonomy mode but focused on structured, scheduled work blocks with timetable integration. Ideal for deep work sessions where GENESIS tackles specific projects or tasks.

Architecture Overview

GENESIS is built on several key components:

  • Cognitive Architecture — The "brain" that orchestrates all decision-making
  • Memory System — Episodic, semantic, and working memory for learning and recall
  • State Manager — Tracks current state (idle, thinking, executing, resting, etc.)
  • Mode Controller — Manages switching between Chat, Autonomy, and Work modes
  • GUI Panel — The user interface you interact with
  • Schedule Manager — Handles timetable-based task scheduling
  • Project Board — Kanban-style project/task management (Backlog → Todo → In Progress → Review → Complete)

Key Features

  1. Multi-mode Operation — Switch between interactive and autonomous modes seamlessly
  2. Smart Scheduling — Time-based task planning with automatic sorting and conflict detection
  3. Project Management — Full kanban board for tracking work across stages
  4. Memory System — Long-term learning from past experiences
  5. Cognitive Load Management — Monitors and adapts to prevent overload
  6. Internal States — Energy, curiosity, frustration, satisfaction tracking for adaptive behavior
  7. Telegram Integration — Remote control via Telegram bot
  8. API Governor — Manages AI API usage to stay within limits

How It Works

GENESIS follows a simple workflow:

  1. You set a goal or ask a question
  2. GENESIS analyzes the request using its cognitive architecture
  3. Depending on the mode, it either responds directly (Chat) or creates a plan and executes it (Autonomy/Work)
  4. Progress is tracked in real-time through the GUI
  5. Results are saved to memory for future reference

Getting Started

  • Chat Mode: Just type in the chat input box and press Enter
  • Autonomy Mode: Set a goal using the "🎯 Set Goal" button, then click the play button ▶
  • Work Mode: Use the same controls as Autonomy mode but with scheduled timetable blocks

See the Modes Comparison guide for detailed differences between modes.
