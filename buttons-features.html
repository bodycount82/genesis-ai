Buttons & Features Guide

Sidebar Navigation (Left Panel)

Main Button (⌂)

Returns you to the main dashboard view. Shows system status, state indicator, cognitive load, API usage, internal states, and mode controls.

Memory Button (🧠)

Opens the memory panel showing GENESIS's stored knowledge:

  • Working Memory — Short-term active memories with importance scores and decay rates
  • Episodic Memory — Past experiences and lessons learned
  • Semantic Memory — Factual knowledge stored long-term

Tracker Button (📋)

Opens the project tracker (kanban board). Shows all tasks organized in columns: Backlog → To Do → In Progress → Review → Complete.

Settings Button (⚙)

Opens the settings panel with all configurable options for GENESIS behavior, API limits, model settings, and system preferences.

Telegram Button (✈)

Toggles the Telegram bot integration. Green dot means the bot is running and you can control GENESIS from Telegram. No dot means stopped.

Top Bar (Main Panel)

State Indicator

Halted🔴 Red ⊗Stopped or errored

Cognitive Load Bar

Horizontal bar filling from left to right. Green (low) → Yellow (medium) → Red (high). Shows exact load percentage.

API Usage Counter

Format: "calls / min / hr / day". Helps you stay within your AI provider's rate limits and cost budgets.

Control Buttons (Top Right)

⚡ OVERRIDE Button (Purple)

Clears rest/sleep states immediately. Does NOT stop currently running tasks. Use when GENESIS is resting but you need it working again now.

⏹ STOP ALL Button (Dark Red)

Nuclear option — stops EVERYTHING. Stops current action in any mode, clears the entire task queue, resets all states to idle. Use for emergency stop when things go wrong.

⏸ STOP Button (Red)

Stops current action gracefully. Stops current task/action, clears the queue, sets state to idle. Use for normal stopping during any operation.

Mode Toggle Row

💬 CHAT MODE Button

Switches GENESIS to interactive chat mode. Waits for your messages, responds one at a time. Best for questions, quick tasks, and conversations.

🤖 AUTONOMY MODE Button

Switches to autonomous goal-directed operation. GENESIS creates its own plan and executes it without intervention. Best for complex multi-step projects.

⚡ WORK MODE Button

Switches to scheduled work mode with timetable blocks. Work is planned into specific time slots in the timetable. Best for structured work sessions with defined start/end times.

▶ Play Button (Green)

Starts autonomy/work mode execution immediately. GENESIS must have a goal set before this will work effectively.

🎯 Set Goal Button

Opens a dialog to enter a new goal for GENESIS. Be specific: "Research Philippines cottage investments and create a report" is better than "do research".

Chat Interface (Bottom of Main Panel)

Message Input Box

Type your message/question here. You can paste text, URLs, and code into the input box.

Tab System

  • Chat Log (Tab 1): Shows clean conversation (messages only)
  • Full Log (Tab 2): Shows everything including thinking, reflections, and system messages

Work Mode Banner (Orange Bar in Tracker)

⚡ Work Mode is Running

Visible only when work mode is active. Shows current work mode status.

⏹ Stop Work Mode Button

Stops the current work mode session. Ends the mission, saves progress, returns to idle.
