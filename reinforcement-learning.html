📚 How Reinforcement Learning Works

🤖 The Agent

The agent is an AI entity that perceives its environment, takes actions, and receives rewards or penalties. Over time, it learns which actions yield the most reward — without being explicitly programmed with the solution.

🌍 The Environment

The environment is everything outside the agent. In our maze, walls are obstacles, the goal gives +10 reward, and pits give -5. The agent starts with zero knowledge and must explore to discover the rules.

🎯 Q-Learning

A value-based RL algorithm that builds a Q-table mapping state-action pairs to expected future rewards. The agent iteratively updates values using the Bellman equation.

Q(s,a) ← Q(s,a) + α[r + γ·max Q(s',a') − Q(s,a)]

🔀 Exploration vs Exploitation

The ε-greedy strategy balances trying new actions (exploration) vs using known good actions (exploitation). High ε early on encourages exploration; as learning progresses, ε decreases to exploit knowledge.

💰 Reward Shaping

Rewards guide learning. Small step penalties (-0.1) encourage efficiency. Large goal rewards (+10) and pit penalties (-5) create clear objectives. This shaping dramatically speeds up convergence.

🧠 Connection to Genesis

Genesis uses similar principles: it explores the solution space, receives implicit feedback from task outcomes, and updates its internal "Q-values" through memory consolidation. Each interaction is a training episode.
