🤖 Autonomy Mode Guide

Autonomy Mode transforms GENESIS from a conversational assistant into an independent worker. It sets its own goals, creates plans, executes tasks step by step, and reports results — all without you needing to guide each step.

How to Start Autonomy Mode

Method 1: Using the GUI

  1. Click the 🤖 AUTONOMY MODE button in the mode toggle row (it will highlight blue)
  2. Set a goal using the 🎯 Set Goal button or type a goal in chat
  3. Click the green ▶ Play button to start

Method 2: Via Chat

Type something like: "Start autonomy mode with the goal of researching and summarizing quantum computing basics"

The Autonomous Loop

Once started, GENESIS enters a continuous cycle:

  1. Goal Analysis — Breaks down the goal into manageable sub-tasks
  2. Planning — Creates an ordered list of actions to complete the goal
  3. Execution — Performs each action (research, code, analyze, create files, etc.)
  4. Progress Tracking — Monitors what's done and what remains
  5. Adaptation — Adjusts the plan if obstacles are encountered
  6. Reporting — When complete, summarizes results

What GENESIS Can Do Autonomously

  • Research topics using web search and file reading
  • Write code and scripts
  • Create documents and reports
  • Analyze data and generate insights
  • Manage project tasks (add, move, complete)
  • Schedule work in the timetable
  • Search and organize files on your system

Cognitive Load Management

While working autonomously:

  • GENESIS monitors its own cognitive load
  • If load gets too high, it may request a rest break
  • Energy levels affect performance — low energy means slower execution
  • GENESIS can enter rest/sleep states automatically if needed

Setting Effective Goals

Good Goal Examples

  • "Research and create a report on the top 5 electric vehicles under $30,000"
  • "Build a Python web scraper that extracts product prices from e-commerce sites"
  • "Analyze my project tasks and reorganize them by priority and deadline"
  • "Create a comprehensive guide to machine learning for beginners"

Goal Tips

  1. Be specific about the output — "Create a markdown report" vs "do research"
  2. Set scope clearly — "For the next 30 minutes, work on..."
  3. Define success criteria — "The goal is complete when you have X deliverables"
  4. Allow flexibility — GENESIS figures out the best approach; tell it the what, not the how

State Indicators

  • Thinking (Cyan): GENESIS is analyzing the goal and planning
  • Executing (Yellow): Actively working through tasks
  • Resting (Orange): Taking a break (may resume automatically)
  • Halted (Red): Stopped due to error or manual intervention

Stopping Autonomy Mode

Graceful Stop (⏸)

  • Stops the current action
  • Completes the current sub-task if possible
  • Returns to idle state, preserves progress made

Emergency Stop (⏹)

  • Immediately halts everything
  • Clears all pending actions
  • Resets to idle state, may leave partial work incomplete

Tips for Effective Autonomy Mode

  1. Set clear goals — The more specific your goal, the better the results
  2. Monitor progress — Watch the Full Log to see what's happening
  3. Don't interrupt unnecessarily — Let GENESIS work through its plan
  4. Check results after completion — Review what was created before setting a new goal
  5. Use follow-up goals — Build on previous autonomous sessions for incremental progress

Troubleshooting

GENESIS keeps stopping: Check if cognitive load is too high, look at the Full Log for error messages, try a smaller more focused goal.

Not making progress: Make the goal more specific, check if GENESIS has access to necessary tools/files, verify API connectivity and rate limits.

Unexpected behavior: Use ⏸ STOP to clear the current state, review the Full Log for what went wrong, adjust your goal or settings as needed.
